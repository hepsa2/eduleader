# 学生依法维权互助
### 《中华人民共和国宪法》
第33条：<br>
**国家尊重和保障人权** <br>
第41条：<br>
**公民有权:** <br>
- 对国家机关和工作人员提出批评、建议
- 进行申诉、控告、检举
- 因违法失职行为受到损害的，有权取得赔偿
### 《中华人民共和国未成年人保护法》
- 第20条：学校不得加重学习负担
- 第21条：保障休息、娱乐、体育时间
